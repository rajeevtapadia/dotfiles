# ToDo-List

A native ToDo list widget for KDE Plasma 6

## Features

- Create and manage tasks
- Task completion tracking
- Drag & drop reordering
- Persistent storage

## Installation

### Prerequisites

- KDE Plasma 6.0 or higher
- Qt 6

## As a Standalone Window

Run the following command to launch as a standalone window:

```bash
  plasmawindowed --statusnotifier de.marc.todo
```
